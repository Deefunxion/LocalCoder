Τέλεια! Gradio εγκαταστάθηκε. Τώρα μπορείς να τρέξεις το Web UI:

  Τρόπος 1: Double-click το .bat file

  Απλά κάνε double-click στο D:\LOCAL-CODER\start_web_ui.bat

  Τρόπος 2: Από Command Prompt

  cd D:\LOCAL-CODER
  start_web_ui.bat

  Τρόπος 3: Απευθείας με Python (αν το .bat δεν δουλεύει)

  cd D:\LOCAL-CODER
  D:\LOCAL-CODER\academicon-agent-env\Scripts\python.exe web_ui.py

  ---
  Μόλις τρέξει, θα δεις:

  ============================================================
  Academicon Code Assistant
  ============================================================

  [1/5] Loading embedding model...
     [GPU] NVIDIA GeForce RTX 5070 Ti
     ...
  [5/5] Ready!
  ============================================================

  Running on local URL:  http://127.0.0.1:7860

  Άνοιξε το browser στο http://localhost:7860 και θα δεις το Gradio UI! 🚀

  Δοκίμασε ένα από τα 3 και πες μου αν ανοίγει!
